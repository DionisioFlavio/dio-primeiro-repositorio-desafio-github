## Comandos



cd (colocando esse comando, junto com o nome da pasta onde deseja acessar, te leva diretamente para lá)

ls ( List, lista onde você esta)

git status (te atualiza do que foi feito, te mostra o status no momento)



Criando Repositório 



Abrir o GitBash ( onde você desejar...Criar uma pasta e abrir direto nela é uma boa)

cd ( com o nome da pasta te leva direto para ela)

mkdir (cria uma pasta dentro dessa outra pasta)

git init 

ls -a ( te mostra arquivos ocultos dento da pasta)

cd.. (voltar 1 níve onde esta)

git add. (adicionar os comandos para validação dentro do seu PC )

git commit  -m ( comita seu desenvolvimento, mandando ele para o servidor do GIT - após o -m * colocar uma mensagem para falar sobre o que foi feito, sempre bom colocar mensagem clara daquilo que se esta fazendo)

git push origin - ( enviar e salvar ele dentro do servidor, sempre fazer esse comando apos comitar, caso contrario não terá atualizado o seu repositório)

git push origin master (não me lembro agora, vai que alguem leia isso e coloque mais codigos que um iniciante possa utilizar ... talvez ate arrume as besteiras que escrevi )
